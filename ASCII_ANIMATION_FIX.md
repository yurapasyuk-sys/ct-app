# 🎬 ASCII Animation Fix

## Проблема
ASCII анимация "Fetching_data..." при переходе на dashboard пропадала слишком быстро (~1 секунда вместо 3+ секунд).

## Причины
1. **sessionStorage блокировал анимацию** - при повторных посещениях анимация полностью пропускалась
2. **Размер текста был слишком маленький** - 120px вместо запланированных 130px
3. **Тайминги не оптимизированы** - 3500ms + 1000ms было избыточно

## Исправления

### ✅ 1. LoadingOverlay.tsx
**Изменены тайминги:**
```typescript
// Было:
const animationDuration = 3500; // 3.5s
const fadeOutDuration = 1000;   // 1s

// Стало:
const animationDuration = 3200; // 3.2s ASCII animation
const fadeOutDuration = 800;    // 0.8s fade out
// Общее время: 4 секунды
```

**Увеличен размер текста:**
```typescript
// Было:
textFontSize={120}

// Стало:
textFontSize={130}
```

### ✅ 2. Dashboard.tsx
**Удалена логика sessionStorage:**
```typescript
// УДАЛЕНО:
useEffect(() => {
  const hasVisited = sessionStorage.getItem('dashboardVisited');
  if (hasVisited) {
    setShowLoading(false); // ❌ Анимация пропускалась!
    setContentOpacity(1);
  }
}, []);

// РЕЗУЛЬТАТ:
// Теперь анимация показывается ВСЕГДА при переходе на dashboard
```

## Итоговые характеристики

| Параметр | Старое значение | Новое значение |
|----------|----------------|----------------|

| **Total Duration** | 4500ms | **4000ms** |
| **Text Size** | 120px | **130px** |
| **sessionStorage** | ✅ Enabled | ❌ Disabled |
